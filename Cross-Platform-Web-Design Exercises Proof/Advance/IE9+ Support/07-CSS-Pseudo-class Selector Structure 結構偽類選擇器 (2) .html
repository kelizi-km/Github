    小結

  1. 結構偽類選擇器一般用於選擇父級里面的第幾個孩子
  2. nth-child 對父元素里面所有孩子排序選擇（序號是固定的） 先找到第n個孩子，然後看看是否和E匹配
  3. nth-of-type 對父元素里面指定子元素進行排序選擇。 先去匹配E ，然後再根據E 找第n個孩子
  4. 關於 nth-child（ n） 我們要知道 n 是從 0 開始計算的，要記住常用的公式
  5. 如果是無序列表，我們肯定用 nth-child 更多
  6. 類選擇器、屬性選擇器、偽類選擇器，權重為 10
